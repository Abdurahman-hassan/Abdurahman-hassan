## 🙋‍♂️ About Me

I'm a **Senior Backend Software Engineer** working in **energy-tech and flexibility trading systems**, where I focus on backend architecture, Kafka-based integrations, task orchestration, and reliable, scalable service design.

I build with **Python, Django, DRF, FastAPI, Flask, Celery, Redis, and PostgreSQL**, applying **Clean Architecture** and **Domain-Driven Design** to keep large production codebases maintainable.

Beyond shipping software, I enjoy **mentoring developers, reviewing code, and turning business requirements into clean technical solutions** — I've supported **250+ developers** through workshops and mentorship.

- 🔭 Currently deep in **event-driven systems, Kafka, and Celery orchestration at scale**
- 🌱 Leveling up in **Kubernetes, Go, platform engineering, and SRE fundamentals**
- 💬 Ask me about **backend architecture, async processing, system design, and API design**
- 🎓 Completing a **Postgraduate Diploma in Computer Science** at Cairo University
- ⚡ Fun fact: I care as much about a clean commit history as I do about clean code
